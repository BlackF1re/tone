## 🎓 Чему мы научились

1. **Автоопределение > Конфигурация** — один бинарник везде работает
2. **Гибридный подход эффективен** — sleep+spinlock = лучше всех
3. **Статистика критична** — без неё не видны реальные проблемы
4. **Сохранение состояния важно** — будь вежлив к соседям по GPIO
5. **Документация = неотъемлемая часть** — код без документации = неполный проект

## 🚀 Как использовать сейчас

### Вариант 1: Простое использование
```bash
# Скопировать бинарник на роутер
scp /root/tone/tone root@192.168.1.1:/usr/bin/

# Использовать
ssh root@192.168.1.1
tone 2000 500
```

### Вариант 2: Из скрипта
```bash
#!/bin/sh
# Звуковой сигнал успеха
tone 2000 100
tone 0 50
tone 2500 100
```

### Вариант 3: В веб-интерфейсе (LuCI)
```lua
os.execute("tone 2000 100")  -- Вызов из Lua
```
